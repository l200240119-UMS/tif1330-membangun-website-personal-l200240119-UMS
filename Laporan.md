# Laporan Tugas #1: Membangun Website Personal - Rei Design

## 1. Deskripsi Proyek
Website ini merupakan platform portofolio digital statis yang dirancang untuk memperkenalkan identitas diri, memamerkan karya UI/UX, dan menyediakan sarana kontak profesional. Website dibangun dengan fokus pada estetika minimalis, performa yang cepat, dan aksesibilitas bagi pengguna.

**Fitur Utama:**
- **Desain Responsif:** Layout yang beradaptasi dengan layar Mobile dan Desktop.
- **Dark Mode Toggle:** Fitur beralih tema menggunakan JavaScript dan LocalStorage.
- **Navigasi Semantik:** Menggunakan struktur HTML5 modern (Header, Nav, Main, Footer).
- **Aksesibilitas:** Kontras warna tinggi dan atribut ARIA untuk pembaca layar.

**Teknologi yang Digunakan:**
- HTML5 (Semantik)
- CSS3 (Flexbox & Grid)
- JavaScript (DOM Manipulation untuk Theme Toggle)
- Google Fonts (Geist)
- FontAwesome Icons
